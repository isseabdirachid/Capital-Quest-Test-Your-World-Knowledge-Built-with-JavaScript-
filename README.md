# Capital Quest: A JavaScript-Powered Geography Challenge

Welcome to **Capital Quest**, an interactive geography quiz built with JavaScript! Test your knowledge of world capitals in a fun and engaging way. This project was designed to help learners of all levels improve their geography skills and challenge themselves with questions about countries and their capitals.
